# ADAM Enterprise Signage System

Enterprise-grade digital signage platform with 9x9 PiP grid, centralized management, and Chromecast support.

## Features
- 9x9 Picture-in-Picture grid display
- Centralized content management
- Device and zone-based scheduling
- Remote casting via Chromecast
- Progressive Web App (PWA) support

## Project Structure
- `admin.html`: Administration panel
- `sender.html`: Casting interface
- `receiver.html`: Chromecast receiver
- `player.html`: Standalone player
- `data/*.json`: Configuration and content data

## Setup
1. Clone the repository
2. Edit JSON files to configure your setup
3. Push changes to deploy
